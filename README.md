# Job Swap 
## What does it do?
Job swap addon meant to be used with Autoduty. Allows you to select classes that you want leveled and set the desired level. After that, you just set autoduty to Auto leveling mode and start job swap. It will automatically level the jobs you designate to the level you set. There's even an option to have it start autoretainer multi mode after it finishes so you're not just idle, but also sending out retainers on all of your characters.
